# DefizAI

An experimental artificial intelligence system.

## Overview

DefizAI is a lightweight AI model focused on exploring minimalistic approaches to human like interaction.

The project explores a simple question:

> How much is actually required for an artificial system to feel natural?

Unlike traditional AI systems, DefizAI does not rely on large datasets, complex architectures, or expensive hardware.

It is built with simplicity as its core principle.

## Features

- Extremely lightweight
- No external dependencies
- Runs locally
- Fast startup time
- Minimal resource usage

## Requirements

- Python 3.x
- computer (preferably)
